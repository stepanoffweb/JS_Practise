# s_scroll	

1.  принцип jQuery, подключение скриптов в html:
    - база для подключения плагинов  base.js - объектв который дабвляем необходимые поля
    - публичные методы open()/close()/setContent()/destroy() и объект с настройками визуализации (title/content/width/closable/... массив кнопок)
    - создаем свой метод insertAfter() у прототипа Element.prototype для добавления своей ноды после конкретной в шаблоне
    - используем Promise для обработки событий двух кнопок модалки (Удалить/ Отмена) 
    - создаем хуки на закрытие/ открытие модалки, чтобы передавать их в опциях с произвольными обработчиками
## Запуск 
	- скачать папку
	- запустить в браузере index.html из нужной директории
